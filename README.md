# learn-opengl

## Tut1: Create a dark window

### Installation and Build instruction (Arch Linux)

* Install GLFW

`sudo pacman -S glfw-x11`

* Install `glad` though [online generator](https://glad.dav1d.de/) and extract to the directory of your repo. 

#### Build

* `mkdir build`
* `cd build`
* `cmake ..`
* `make`
* `cd ..`
* `./tutorial`
